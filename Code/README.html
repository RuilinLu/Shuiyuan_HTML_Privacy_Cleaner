水源 HTML 隐私清理工具 V5

这是一个可完全离线运行的 Windows 工具，用来清理用 SingleFile 保存的水源/Discourse HTML 快照。V5 的全匿名模式直接在原始 SingleFile/Discourse HTML 上就地匿名化，尽量保留原版论坛布局，而不是提取发言后重写一个新页面。

能清理什么

  • SingleFile 保存时间注释、可见水印、隐藏低透明度整页水印
  • 右上角当前登录用户头像、账号入口、currentUser、未读/新帖计数、阅读状态
  • Thunderbit、沉浸式翻译等扩展残留
  • CSRF token、主题激活、tracking、用户手动填写的审核关键词
  • 全匿名模式下的站点域名、站点品牌、外链、预加载 JSON、公开用户标识、头像、徽章、头衔和 flair

两种模式

仅清除当前保存者 / 登录者信息

只删除保存者自己的水印、登录态、头像、ID、昵称和保存痕迹。这个模式会保留页面原有站点名称、帖子里其他公开用户、公开头像、用户徽章、公开链接和论坛结构。

全匿名模式

在原始 Discourse/SingleFile 框架内就地改写，把同一用户的昵称、用户名、@用户名、用户链接和头像信息尽量统一替换为同一个 用户1、用户2，并删除用户徽章、站点外链和隐藏预加载数据。

界面区域

  1. 输入 HTML：选择原始 SingleFile 保存的 .html 或 .htm 文件。
  2. 输出 HTML：选择清理后的保存位置，切换模式后会自动建议文件名。
  3. 审核关键词：填写用户名、ID、头像编号、时间戳、昵称或可疑字符串。
  4. 模式：选择个人模式或全匿名模式。
  5. 仅分析：只扫描输入文件，不写新 HTML。
  6. 清理并审核：执行清理，写出 HTML，并生成审核报告。
  7. 打开输出位置：在资源管理器里定位输出文件。
  8. 允许覆盖输出文件：勾选后允许覆盖已有结果。
  9. 报告区：显示清理前检测、实际移除和清理后审核结果。
  10. 底部状态栏：显示当前阶段，例如就绪、清理中、清理完成或出错。

进度窗口

绿色进度条是真实百分比，只从左到右走一遍；窗口会显示当前百分比、处理阶段和已运行秒数。全匿名模式处理大文件时可能较慢，但界面会持续显示进度。

推荐流程

  1. 在 Edge 中打开目标页面，并用 SingleFile 保存为单个 HTML。
  2. 打开对应架构的 ShuiyuanHtmlPrivacyCleaner.exe。
  3. 选择输入 HTML 和输出 HTML。
  4. 选择清理模式，并按需填写审核关键词。
  5. 先点 仅分析，再点 清理并审核。
  6. 打开输出 HTML 确认页面能正常显示，只分享清理后的文件和报告。

命令行模式

.\ShuiyuanHtmlPrivacyCleaner.exe --clean "input.html" "output.html" `
  --mode full `
  --term "用户名" `
  --term "用户ID" `
  --report "report.txt" `
  --overwrite

可用模式为 personal 和 full。路径中有空格时必须加英文双引号。

V5 验收

  • Anthropic 样本全匿名：Edge 实际打开后 topic-post=21，cooked=21，页面不是空白。
  • 国赛样本全匿名：Edge 实际打开后 topic-post=62，cooked=62，页面不是空白。
  • 全匿名输出中保存者关键词、站点域名、公开用户标识、用户徽章节点、隐藏预加载数据、站点外链、CSRF、SingleFile 保存时间、水印、隐水印均审核为 0。
  • 个人模式按设计保留公开站点和其他用户信息，但会清除保存者个人信息和水印。

版本命名

  • APP/V1：第一版个人信息清理工具。
  • APP/V2：加入双模式雏形。
  • APP/V3：加入进度窗口。
  • APP/V4：修复中文和进度体验，曾使用静态重建方案。
  • APP/V5：全匿名改为在原始 Discourse/SingleFile 框架内就地匿名化，并修复空白页问题。
